# Obsidian Swatch

Visualize colours inside your document.

## Inline Swatch
Add a 7-character hex-encoded (`#xxxxxx`) color to an inline code block to render a cute swatch by its side.

## Palette Swatches
You can render a series of swatches using a markdown code block set to the `colors` language.

````md
```colors
#9381ff
#b8b8ff
#f8f7ff
#ffeedd
#ffd8be
```
````
